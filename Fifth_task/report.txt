Отчет о задаче валидации генераторов случайных чисел
----------------------------------------------------------------------
Цель работы:
Реализовать статистические тесты для проверки гипотезы H0
(последовательность является выборкой из независимых равномерно
распределенных величин U(0,1)).
Согласно домашнему заданию необходимо:
1. Проверить гипотезу о соответствии распределения ожидаемому (KS, Chi2).
2. Протестировать свойства независимости (Автокорреляция).
3. Построить пример плохого генератора, который успешно проходит
   тесты на моменты или равномерность, но проваливает тесты на независимость.

ОПИСАНИЕ ИССЛЕДУЕМЫХ ГЕНЕРАТОРОВ
-----------------------------------
- Стандартный вихрь Мерсенна (std::mt19937), используемый как эталон.
- BadGenerator реализован как линейный счетчик с малым
  шагом: x_{n+1} = (x_n + step) mod 1. Данный алгоритм порождает
  квазислучайную последовательность.

МЕТОДОЛОГИЯ ТЕСТИРОВАНИЯ
---------------------------
Реализованы следующие статистические критерии:

1. Критерий Хи-квадрат:
Проверка равномерности распределения по отрезку (bins=100).
Статистика Chi2 = sum((O_i - E_i)^2 / E_i).
Для истинно случайной величины Chi2 должно быть близко к (bins - 1).

2. Критерий Колмогорова-Смирнова:
Проверка близости эмпирической функции распределения к
теоретической.
Статистика D_n = sup|F_n(x) - F(x)|.

3. Тест на автокорреляцию:
Проверка независимости соседних элементов. Вычисляется коэффициент
корреляции Пирсона между x_i и x_{i+lag}. Для случайных чисел R -> 0.

АНАЛИЗ РЕЗУЛЬТАТОВ
-------------------------------------------------

1. Chi-Squared Test:
- MT19937: Значения распределены в диапазоне [70, 140], что соответствует
  нормальной стохастической природе генератора.
- BadGenerator: Статистика в точности равна 0.0.
Вывод: Генератор слишком равномерен. Согласно слайду 9 из лекции, p-value
в данном случае близко к 1, что указывает на детерминированную природу.

2. Kolmogorov-Smirnov Test:
- MT19937: Расстояние D_n составляет ~0.01.
- BadGenerator: Расстояние D_n стремится к 0, т.е. практически полное совпадение.
Вывод: Квазислучайная последовательность заполняет отрезок [0, 1] идеально
равномерно, обманывая критерий Колмогорова.

3. Autocorrelation Test:
- MT19937: Значения сосредоточены около 0.0 с высокой независимостью.
- BadGenerator: Значение коэффициента корреляции равно 1.0.
Вывод: Полный провал теста. Коэффициент 1.0 означает жесткую линейную
функциональную зависимость x_{n+1} от x_n.

ЗАКЛЮЧЕНИЕ
-------------
В ходе работы продемонстрирована ключевая проблема валидации ГСЧ:
генератор может обладать идеальными свойствами равномерности; проходить
тесты Chi2 и KS с результатами лучше, чем у истинно случайных чисел;
но при этом быть абсолютно предсказуемым.

Подобранный BadGenerator подтверждает тезис из
лекции о необходимости многоуровневого тестирования:
1. Равномерность распределения - пройдено.
2. Независимость элементов - не пройдено.
