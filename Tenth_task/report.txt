Оптимизация генераторов случайных чисел на примере векторизации minstd_rand

Процессор:            Intel Core i7-1260P
Тактовая частота CPU: 2.1 ГГц
Кэш L1d (P-core):     48 KiB (8-way associative)
Кэш L2 (P-core):      1.25 MiB
ОС:                   Linux Mint
Компилятор:           g++ (GCC) 13.4.0
Флаги компиляции:     -O3 -fopenmp
Метрика измерений:    CPE (Cycles Per Element)

РЕАЛИЗАЦИЯ И ОПТИМИЗАЦИЯ
В ходе работы были применены следующие техники оптимизации:

        Векторизация. Реализован переход к векторному представлению состояния
      генератора (размер вектора dimension = 8). Константа перехода Ak = a^k mod m вычисляется
      заранее, что позволяет генерировать вектор случайных чисел за одну итерацию.

        Быстрое вычисления модуля. Вместо дорогостоящей операции деления использовано
      свойство числа Мерсенна 2^31 - 1. Применена оптимизация на базе
      битовых сдвигов и AND: (y >> 31) + (y & 0x7FFFFFFF).

        Параллелизация. Использована библиотека OpenMP для распределения нагрузки
      между ядрами CPU.

        Поддержка Skip-ahead. Для обеспечения независимости потоков реализован
      алгоритм быстрого прыжка по последовательности block-splitting. Каждый
      поток инициализирует свое состояние независимо, не вызывая конфликтов памяти.

        Локальность данных. Генерация производится блоками по 512 элементов,
      что гарантирует эффективное использование L1d-кэша процессора.

ТЕСТИРОВАНИЕ КОРРЕКТНОСТИ
Проведено сравнение выходной битовой последовательности реализованного векторного
генератора с эталонной реализацией std::minstd_rand из стандартной библиотеки C++,
кастомный генератор проверку прошел.

РЕЗУЛЬТАТЫ БЕНЧМАРКА ВЫЧИСЛЕНИЕ ЧИСЛА PI
Для тестирования производительности и статистических свойств использован
метод Монте-Карло на 100 млн итераций.

Vector RNG Throughput: 3.0033 CPE +- 0.0573536


АНАЛИЗ РЕЗУЛЬТАТОВ
        Абсолютная ошибка (8.68e-05) находится в пределах доверительного
      интервала трех сигм (4.92e-04). Это подтверждает статистическую
      корректность работы векторизованного генератора и отсутствие
      систематического смещения при параллельной обработке.

        Время выполнения составило менее 0.6 секунды для
      100 млн точек, что говорит о высокой эффективности векторизации
      и минимальных накладных расходах на управление потоками.

Реализованный генератор соответствует требованиям по точности и корректности.
Примененные оптимизации позволили достичь высокой скорости генерации
распределений, пригодной для использования в приложениях метода Монте-Карло.
